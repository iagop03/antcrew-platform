Cómo funciona

De commit a revisión aprobada, sin fricción

Un ciclo de revisión completo en tres pasos. Tu sistema llama a la API; antcrew hace el resto.

⬆︎
Paso 01

Envía el run

Tu pipeline llama a POST /run/ con el código y el contexto. antcrew crea una tarea y la encola.

⚡
Paso 02

Los agentes revisan

El equipo de agentes analiza el código, detecta problemas, propone cambios y genera un informe estructurado.

✓
Paso 03

Tú apruebas

Con HITL activo, el agente espera tu confirmación por Slack, Telegram o API antes de continuar. Sin tu OK, no avanza.

Pago por uso

Cada run procesa tokens de modelos de lenguaje. Te cobramos ese coste a precio de proveedor más una tarifa de servicio. Si no ejecutas runs, no pagas nada.

Coste LLM precio real del proveedor Anthropic / OpenAI según el modelo configurado
Tarifa de servicio 200% sobre el coste LLM Infraestructura, orquestación y soporte — pagas 3× el coste real del modelo
Cuota mensual $0 Sin mínimos ni compromisos

Preguntas frecuentes

Lo que suelen preguntar

¿Cómo se calcula exactamente el coste de un run?
Cada run genera llamadas a modelos de lenguaje. Medimos los tokens de entrada y salida, los multiplicamos por la tarifa del proveedor (Anthropic o OpenAI según tu configuración) y añadimos nuestra tarifa de servicio. El coste exacto de cada run queda registrado en cost_usd y se acumula en total_cost_usd del workspace. Puedes consultarlo en cualquier momento via API o panel.
¿Hay algún mínimo mensual o cuota de suscripción?
No. Si no ejecutas runs, no hay ningún cargo. No hay cuota base, no hay compromiso de permanencia ni penalización por baja actividad.
¿Qué incluye el trial y cuánto dura?
El trial da acceso completo a todas las funcionalidades durante 14 días, sin tarjeta de crédito. Al terminar, tu cuenta pasa automáticamente a pago por uso si introduces un método de pago, o queda bloqueada hasta entonces. La duración exacta del trial puede variar según la campaña o promoción con la que te registres.
¿Mi código pasa por proveedores de IA externos?
Sí. Las revisiones se procesan mediante llamadas a la API de Anthropic y/u OpenAI. Estos proveedores tienen sus propias políticas de retención de datos. Recomendamos no enviar código que contenga secretos, contraseñas o datos altamente confidenciales como parte del contexto del run. Consulta nuestra política de privacidad para más detalle sobre los subencargados de tratamiento.
¿Puedo establecer un techo de gasto?
Sí. Cada workspace tiene un campo budget_usd. Cuando el gasto acumulado lo alcanza, los nuevos runs se bloquean con estado budget_exceeded. Puedes ajustar el presupuesto en cualquier momento sin interrumpir los runs en curso.
¿Puedo cancelar en cualquier momento?
Sí. Cancela desde el panel de control o enviando un email a hola@antcrew.dev. Los runs en curso finalizan normalmente. No hay penalización ni período de aviso.
